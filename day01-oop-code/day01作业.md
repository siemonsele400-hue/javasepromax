1. static可以用来修饰什么？各自有何特点？和应用场景？

* **修饰成员变量（类变量）：**
* 
**特点**：属于类，内存中仅一份，被所有对象共享 。

* 
**场景**：适合存储全局共享的数据，如系统在线人数、学校名称 。

* **修饰成员方法（类方法）：**
* 
**特点**：直接用类名调用，不依赖对象 。

* 
**场景**：常用于编写工具类方法（如 `Math.abs()`） 。
